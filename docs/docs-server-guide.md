# 文檔服務器

## 功能

- 📚 在線瀏覽所有 markdown 文檔
- 🔍 文檔列表自動生成
- 📱 響應式設計，支援手機瀏覽
- 🎨 美觀的 markdown 渲染

## 使用方式

### 安裝依賴

```bash
npm install
```

### 啟動文檔服務器

```bash
npm run docs
```

服務器將在 http://localhost:3001 啟動

### 功能說明

1. **首頁**：顯示所有 `.md` 文檔的列表
2. **文檔查看**：點擊任意文檔即可在線查看
3. **自動渲染**：markdown 內容自動轉換為 HTML
4. **導航**：每個文檔頁面都有返回列表的連結

## 技術實現

- **後端**：Express.js
- **Markdown 解析**：marked
- **樣式**：內嵌 CSS，響應式設計
- **端口**：3001（避免與其他服務衝突）

## 文檔結構

```
docs/
├── *.md                    # 所有 markdown 文檔
├── docs-server.js          # 文檔服務器
```

## 特色

- ✅ 零配置，自動掃描 docs 目錄
- ✅ 支援中文檔名和內容
- ✅ 美觀的程式碼高亮
- ✅ 表格和引用塊樣式
- ✅ 手機友好的響應式設計
