解決方案（按推薦程度排序）

1 載入 Google Fonts Noto CJK（最推薦）

Noto 字型家族專為解決 tofu（豆腐塊）問題而設計，覆蓋幾乎所有 Unicode 字符，包括 Extension B/C/D/E/F。透過 Google Fonts 在線載入，Android Chrome 亦可正常渲染。

<link rel="preconnect" href="https://fonts.googleapis.com"> <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin> <link href="https://fonts.googleapis.com/css2?family=Noto+Serif+TC:wght@400;700&display=swap" rel="stylesheet"> <!-- CSS --> body { font-family: 'Noto Serif TC', 'Noto Sans TC', serif; }

2 完整 font-family Fallback Stack

即使不用 Google Fonts，提供完整的字型 fallback 鏈，讓瀏覽器依序嘗試可用字型。各平台有不同的 CJK 字型，堆疊越完整，成功機率越高。

font-family: 'Noto Serif TC', /* Google Fonts，最廣覆蓋 */ 'Noto Sans TC', /* Google Fonts 備用 */ 'Source Han Serif TC', /* Adobe 思源宋體 */ 'Source Han Sans TC', /* Adobe 思源黑體 */ 'PingFang TC', /* macOS / iOS */ 'Microsoft JhengHei', /* Windows */ 'WenQuanYi Zen Hei', /* Linux */ 'Noto Sans CJK TC', /* Android / Linux 原生 */ serif;

3 自行托管 Noto CJK 字型（離線/內網方案）

若不能使用 Google Fonts，可下載 Noto Sans/Serif CJK 字型檔，透過 @font-face 自行托管。注意字型檔很大（數十 MB），建議使用子集化工具（pyftsubset）只保留需要的字符。

/* 子集化後自托管 */ @font-face { font-family: 'NotoSubset'; src: url('/fonts/NotoSerifCJKtc-subset.woff2') format('woff2'); unicode-range: U+2137C; /* 只載入這個字 */ } /* pyftsubset 指令（需安裝 fonttools） */ # pyftsubset NotoSerifCJKtc-Regular.otf \ # --unicodes="U+2137C" \ # --flavor=woff2 \ # --output-file=subset.woff2

4 Canvas 轉圖片（終極保底，100% 相容）

若字型方案仍無法解決，可用 Canvas 在已載入字型的環境中繪製字符，再轉為 PNG 圖片顯示。此方案完全相容所有瀏覽器，但失去文字可選取、可複製的特性。

// 等字型載入後再繪製 document.fonts.load("48px 'Noto Serif TC'").then(() => { const canvas = document.createElement('canvas'); canvas.width = 80; canvas.height = 80; const ctx = canvas.getContext('2d'); ctx.font = "64px 'Noto Serif TC', serif"; ctx.fillStyle = '#8b2500'; ctx.textBaseline = 'middle'; ctx.fillText('\uD844\uDF7C', 8, 42); // 代理對 const img = document.createElement('img'); img.src = canvas.toDataURL(); img.alt = '𡍼'; document.body.appendChild(img); });

5 確保 HTML 正確設定（基礎必要條件）

無論用哪種方案，以下基礎設定必須正確，否則在某些瀏覽器會強制使用系統預設字型覆蓋 CSS 設定。

<!-- 1. DOCTYPE 必須是 HTML5 --> <!DOCTYPE html> <!-- 2. 字符集必須是 UTF-8，且要放在 <head> 最前面 --> <meta charset="UTF-8"> <!-- 3. 語言標記 --> <html lang="zh-Hant"> <!-- 4. 儲存檔案時確保用 UTF-8 編碼（無 BOM） -->
