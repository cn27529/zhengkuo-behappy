問題根因 & 解決方式

為什麼 Google Fonts 的 Noto Serif TC 不夠？
Google Fonts 提供的 Noto TC 版本是子集化過的，只包含常用繁體字符， 刻意去掉了 Extension B/C/D 等罕見漢字，以減少字型檔大小。 完整版的 Noto Sans/Serif CJK 有數十 MB，Google Fonts 不提供完整版的 web 載入。


✓ 本頁的解法：用 Python fonttools 生成一個只含「𡍼」一個字的迷你 woff2 字型（416 bytes）， 以 base64 直接嵌進 HTML，完全離線，任何瀏覽器都能載入。

複製到你的項目 — CSS @font-face

把以下 CSS 加到你的樣式表，即可在任何頁面使用這個字型：

如何自行生成其他罕見字的字型子集

若你有其他 Extension B 字符需要顯示，先取得完整 Noto CJK 字型，再用 pyftsubset 子集化：

# 1. 安裝工具 pip install fonttools brotli # 2. 下載完整 Noto Sans CJK（需支援 Extension B 的版本） # 從 https://github.com/notofonts/noto-cjk/releases 下載 # NotoSansCJKtc-Regular.otf（約 16MB） # 3. 子集化，只保留需要的字符 pyftsubset NotoSansCJKtc-Regular.otf --unicodes="U+2137C" --flavor=woff2 --output-file=rarechar.woff2 # 4. 轉為 base64 嵌入 HTML（避免額外網路請求） python3 -c " import base64 data = open('rarechar.woff2','rb').read() print(base64.b64encode(data).decode()) "
