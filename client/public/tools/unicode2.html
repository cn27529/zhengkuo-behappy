💡 为什么Android Chrome无法显示而macOS Chrome正常？

• macOS 内置了「苹方」或「宋体-简」等字体包含部分扩展B区字形，或者macOS Chrome会调用「Apple Color Emoji」及后备字体支持较好。
• Android 系统的默认字体（如 Roboto, Noto Sans 普通版）通常只涵盖常用CJK基本区（U+4E00~U+9FFF），不包含扩展B区字符（如𡍼）。很多厂商定制字体裁剪了扩展区。
• 解决方案：通过 CSS 指定包含扩展B区的字体 (如 Noto Sans CJK SC/TC，或花园明朝)，或者使用Web字体动态加载。
