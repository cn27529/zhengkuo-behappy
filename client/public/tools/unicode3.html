罕見字顯示測試 - 𡍼 - gemini

測試字元：𡍼 (U+21374)

1. 預設系統字型 (可能會在 Android 失敗):
𡍼
2. 使用 Noto Sans TC (網路字型 - 建議方案):
𡍼

技術說明：為何 Android 看不到？

  • 編碼範圍：「𡍼」屬於 CJK Extension B。這類字元超出了常用字集（如 Big5 或常用國字標準字體）。
  • 系統差異：macOS 和 iOS 通常內置較完整的字庫（如「苹方」）；而 Android 為了節省空間，預設的 Roboto 或 Noto Sans CJK 可能只包含基本平面（BMP）的文字。
  • 解決方法：透過 CSS 的 @font-face 引入支援擴展區的字型檔。
